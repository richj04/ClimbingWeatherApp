#Climbing Weather App 

-note this is my first App
-make sure to type spellings perfect

This app will tell you when you can climb outdoors based on ideal conditions:

    -default settings for "ideal conditions":
    tempMin = 40.0
    tempMax = 65.0
    humidityMax = 70.0
    badWeather = ('Rain','Snow')

Requirements to run this app:
make sure python is installed,
then run this line: pip install -r requirements.txt

You also need to get your own api key and make a file called .env that contains API_KEY=your key here
-->https://home.openweathermap.org/api_keys
